# Pharmaceutical Care App

A Python console application for managing patient records and medicine reminders in a pharmaceutical care system.

## Features

- ✅ **Add Patient** - Register new patients with their medical information
- ✅ **View Patients** - Display all patient records
- ✅ **Search Patient** - Find a patient by name
- ✅ **Medicine Reminder** - Set medicine reminders with time
- ✅ **User-Friendly Menu** - Easy-to-navigate console interface

## Installation

### Prerequisites
- Python 3.7 or higher
- pip (Python package manager)

### Setup

1. Clone the repository:
```bash
git clone https://github.com/alihaiderarshad506-code/Maax-Pharma-Care.git
cd Maax-Pharma-Care
```

2. Run the application:
```bash
python main.py
```

## Usage

Once you run the application, you'll see the main menu:

```
================================
 PHARMACEUTICAL CARE SYSTEM 
================================
1. Add Patient
2. View Patients
3. Search Patient
4. Medicine Reminder
5. Exit

Enter Choice: 
```

### Menu Options

1. **Add Patient** - Enter patient details (name, age, disease, medicine, dosage)
2. **View Patients** - Display all registered patients
3. **Search Patient** - Find a patient by entering their name
4. **Medicine Reminder** - Set a medicine reminder with time
5. **Exit** - Close the application

## Example

```
--- Add New Patient ---
Enter Patient Name: John Doe
Enter Age: 45
Enter Disease: Diabetes
Enter Medicine Name: Metformin
Enter Dosage: 500mg twice daily

Patient Added Successfully!
```
